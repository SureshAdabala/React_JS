Rules of creating of JSX :
1. Single Parent Element-> Only one element(div) in return the jsx Code.
2. class in the place of className. Because class is reserved name.
3. the Attributes are in CamelCase.
4. Enclosing correct.
5. Javascript code is write in flower brackets.
6. Style will gives in Object format.
